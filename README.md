## 🧠 About Me

I'm a 19-year-old self-taught developer from Ndola, Zambia, building AI-powered web applications that solve real problems for African communities. I started coding in April 2024 and have shipped real, deployed products ever since — currently building entirely from a smartphone using Termux.

I'm the founder of **Elsoft**, a personal tech brand built on one principle:
> "We solve people's problems with technology in an open way."

I'm joining **Copperbelt University** in November 2026 for a **BSc in Computer Science**, using it to deepen my engineering foundation rather than waiting to start building.

Every project I ship follows a structured engineering discipline I call **code.josh.claude**: architecture-first thinking, small verified increments, and full accessibility, responsiveness, and SEO compliance from the start — never bolted on afterward.

## 📦 The Elsoft AI Portfolio

Six AI-powered tools built end to end — architecture, implementation, testing, and deployment — aimed at solving everyday problems for Zambian communities. All built from a smartphone using Termux.

| Status | Project | Description | Stack |
|--------|---------|--------------|-------|
| ✅ Live | **[FarmerAI](https://farmer-ai-delta.vercel.app/)** | Stock and harvest tracker for smallholder farmers, with an AI chat assistant for real-time farming advice | HTML5 · CSS3 · Vanilla JS · Groq (LLaMA 3.3) |
| ✅ Live | **[Daily Word](https://daily-word-orpin.vercel.app/)** | A quiet daily devotional — KJV verse of the day with an AI-generated personal reflection | Next.js · TypeScript · Tailwind · Groq |
| ✅ Live | **[GradeBoost Zambia](https://gradeboost-zambia.vercel.app/)** | Study tool for Zambian students — explains a topic at any grade level, then quizzes understanding | Next.js · TypeScript · Tailwind · Groq |
| ✅ Live | **[ZambianRecipes AI](https://zambian-recipes-ai.vercel.app/)** | Enter ingredients you have at home, get authentic traditional Zambian recipes with full instructions | Next.js · TypeScript · Tailwind · `useReducer` · Groq |
| ✅ Live | **[ShopBot Ndola](https://shopbot-ndola.vercel.app/)** | Conversational AI shopping assistant for a local retail catalogue | Next.js · TypeScript · Zustand · React Hook Form · Groq |
| 🔜 In Progress | **KopalaMove** | Flagship project — transport and logistics helper for the Copperbelt, combining every pattern learned across the portfolio | Next.js · TypeScript · Full-stack architecture |

Every project in this table is fully responsive, accessible (WCAG AA), and SEO-optimised from first commit — not retrofitted.
